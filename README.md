## 💽 Installation Guide

**Requirements:**
- Node v14
- TypeScript
- ffmpeg
- MongoDB
- Make & gcc (in order to build sodium)

Follow installations guides for your OS, and then install all dependencies:<br/>
`yarn` or `npm i`
